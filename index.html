FAQ

Help, the server isn't connecting! 1. Re-check the Padlet for server URL updates; we rotate them on a regular basis as a security measure to prevent abuse.
2. Ensure that the server URL starts with "wss://" rather than "https://", and that it ends with "/server"
3. Talk to whoever gave you the password to the server URL access page.
Can you upgrade the server? Unfortunately, previously attempted version upgrades have resulted in frequent server crashes. Although a version upgrade is highly unlikely, we are still working towards upgrading to v1.8.
Can I connect to a vanilla 1.5 server? No, the client only supports websockets and offline-mode-enabled servers. Instead, use a normal client.
